# Jarvis #
## Survey REST API ##

Provides a REST API for creating and managing surveys

### Setup Project for Development ###

1. Install Node
2. Install Mongo
3. `cd survey-service`
4. Run `npm install`

### Run survey-service ###

1. Run `mongod`
2. Run `npm start`

## Run Integration Tests ##

1. Run `sh ./run-tests.sh`

## Docker Deploy ##

1. `docker-compose up`
